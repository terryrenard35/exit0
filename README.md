# 🚀 Exit0
> **The Open Source Knowledge Engine for IT Professionals.**
> *Because every problem should end with an `exit 0`.*

---

## 💡 Le Concept
**Exit0** est une plateforme d'entraide ultra-rapide conçue pour les pros de l'IT. Notre mission : vous permettre de trouver la solution à un bug applicatif en moins de 10 secondes.

- **App-Centric :** Recherche ciblée par application et version.
- **Real-time Help :** Messagerie instantanée entre experts pour résoudre les incidents.
- **Community Driven :** Entièrement Open Source.

## 🏗️ Stack Technique
- **Cloud :** Google Cloud Platform (GCP)
- **Runtime :** Cloud Run (Serverless)
- **Network :** Isolated VPC for security
- **Database :** Firestore (NoSQL)
- **IaC :** Terraform
